# Portfolio Template — Dark (Video Hero)

Dark-themed portfolio with video hero background, experience carousel, project case studies, and achievements section.

Works for any role — replace all placeholder text and assets with your own content.

## Customize

| What | Where |
|------|-------|
| Name, title, bio, contact | `src/components/` |
| Profile photo | `public/placeholder.svg` |
| Company logos & project images | `/placeholder.svg` references in components |

## Run locally

```sh
npm i
npm run dev
```
